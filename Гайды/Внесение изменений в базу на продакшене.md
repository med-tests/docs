*Когда нужно добавить, переименовать или удалить колонку или типа того*

1. Написать необходимый скрипт в файле `scr/db/db-action.js`
2. Запушить
3. Зайти на сервер, перейти в папку бэка `cd /back`
4. Сделать бэкап базы продакшена [[Внесение изменений в базу на продакшене]]
5. Стянуть изменения `sudo git pull`
6. Запустить скрипт изменений командой `npm run base_action`
7. Проверить, что новая функциональность работает и старая не сломалась
8. Перенести тэг на текущий коммит и запушить его [[Работа с гит-тэгами]]
9. Удалить бэкап
