
Функции по умолчанию юзаешь с стора, все остальное через storeToRefs для удобства, туды сюды

Если деструктурировать store напрямую
``` js
const { user, posts } = useUserStore()
```
то переменные `user` и `posts` потеряют реактивность — они станут обычными значениями на момент деструктуризации и не будут обновляться при изменениях в store.

``` js
const userStore = useUserStore()

// Извлекаем реактивные ссылки на state и getters
const { name, email, fullName } = storeToRefs(userStore)

// Для actions деструктуризация без storeToRefs безопасна
const { updateUser } = userStore
```
 теперь можно использовать `name`, `email`, `fullName`, и они будут реактивно обновляться
