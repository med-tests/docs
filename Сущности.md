[[Med-tests]]
##### Анализ (test)
- **id**\* - *INTEGER* - id анализа. Устанавливается базой.
- **title**\* - *TEXT* - пользовательское название анализа
- **userId**\* - *INTEGER* - id пользователя, которому принадлежит анализ. Не отправляется на фронт, используется для поиска результатов анализа.
- **normalFrom** - *INTEGER* - пользовательское значение нижней границы нормы для анализа
- **normalTo** - *INTEGER* - пользовательское значение верхней границы нормы для анализа
- **isHidden** - *INTEGER* - определяет, отрисован график или нет. 0 - показать, 1 - скрыть.
- **showFrom** - *TEXT* - дата в формате `YYYY-MM-DD`, если есть хоть один результат анализа или пустая строка. Определяет *с* какой даты включительно выводятся результаты анализа на графике. По умолчанию - дата самого раннего анализа. Может переопределяться пользователем. Если у анализа нет результатов, будет пустой строкой.
- **showTo** - *TEXT* - дата в формате `YYYY-MM-DD`, если есть хоть один результат анализа или пустая строка. Определяет *до* какой даты включительно выводятся результаты анализа на графике. По умолчанию - дата самого раннего анализа. Может переопределяться пользователем. Если у анализа нет результатов, будет пустой строкой.
- **results**  - *ARRAY* - массив результатов анализа.
- **status** - *INTEGER* - Указывает, удален ли анализ. 0 - удален, 1 - активен. Не отправляется на фронт, но по нему фильтруются анализы, отправляемые на фронт.

##### Результат анализа (result)
- **id**\* - *INTEGER* - id результата анализа. Устанавливается базой.
-  **userId**\* - *INTEGER* - id пользователя, которому принадлежит результат анализ. Не отправляется на фронт, используется для поиска результатов анализа.
- **date**\* - *TEXT* - дата в формате `YYYY-MM-DD`. Пользовательская дата сдачи анализа.
- **value**\* - *INTEGER* - Пользовательский результат анализа. Может быть 0.
- **status** - *INTEGER* - Указывает, удален ли результат анализа. 0 - удален, 1 - активен. Не отправляется на фронт, но по нему фильтруются результаты, отправляемые на фронт.
- **test_id**\* - *INTEGER* - id анализа к которому принадлежит результат. Не отправляется на фронт, используется для поиска результатов анализа.

##### Юзер (user)
- **id**\*  - *INTEGER* - id пользователя
- **login\*** - *VARCHAR(255) UNIQUE* - уникальное имя пользователя
- **password\*** - *VARCHAR(255)* - пароль
- **status** - *INTEGER* - Указывает, удален ли пользователь. 0 - удален, 1 - активен. 
- **date_create** - *DATETIME* - Таймштамп даты создания пользователя
