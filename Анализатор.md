### Описание
Приложение для отслеживания динамики показателей. На самом деле, это могут быть любые показатели, не только связанные со здоровьем. Но только 0 и положительные числа.

### Функционал
\* - не нужно делать для MVP

Юзер должен иметь возможность:
- видеть норму для каждого показателя
- видеть динамику показателей (графики)
- выбирать период за который выводится график
- соединять несколько показателей на одном графике
- видеть выходы показателей за пределы нормы
- настроить размер шрифта\*
- скрывать и делать видимыми графики
- менять порядок расположения графиков
- добавление, редактирование и удаление данных для графиков

### Сущности
###### Список показателей. 
Каждый элемент должен:
- иметь название
- при клике перематывать страницу к своему графику
- менять порядковый номер графиков\*
- иметь возможность скрыть и показать свой график\*
- сохранять предпочтения просмотра после перезагрузки (local storage?)\*


###### График должен:
- иметь название по параметру (например, "Гемоглобин, Hb")
- иметь ось времени, где указаны даты сдачи анализов
- иметь ось значений, где указаны значения результатов анализов
- ось значений должна быть +- 10, относительно наибольшего и наименьшего показателей за период
- иметь легенду, где указаны границы нормы для показателя
- иметь возможность выбрать период для отображения
- иметь возможность выбрать период для отображения для всех графиков
- у каждой точки должна быть информация: какой результат, за какую дату, в норме ли он (цвет точки. если зеленая -ок, если красная - нет)
- по некоторым показателям возможны сводные графики (например, для гормонов щитовидной железы)
- иметь возможность быть открытым на весь экран\*
-  иметь возможность скрыть себя (сделать обратно видимым можно только из списка показателей)\*
