`PATCH` `/api/position/:id`
**id** - id анализа, целое положительное число

**Описание:**  обновляет `position` анализа и всех затронутых перемещением анализов. Возвращает id анализа.

**Тело запроса**
``` json
{
	"newPosition": 10, // обязательное
	"oldPosition": 12, // обязательное
}
```

**Успешный ответ (200):**
``` json
{
	"id": 9
}
```
