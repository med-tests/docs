[[Med-tests]]

##### Формат контролируемой ошибки
`status 200`
``` json
{
	"error": true, // обязательное
	"message": "<Описание ошибки, которое будет выведено в тостер>", // обязательное
	"error_code": 1,
}
```

error_code 1 - ошибка авторизации (устарел токен, пользователь не найден, нет токена)

##### Запросы
- [[Логин]] `/api/login`
- [[Регистрация]]  `/api/register`
- [[Получить список всех анализов]]  `api/get-tests`
- [[Добавить анализ]] `api/add-test`
- [[Редактировать анализ]]  `/api/edit-test/:id`
- [[Удалить анализ]] `/api/delete-test/:id`
- [[Изменить порядковый номер анализа в списке  и в графиках анализов]] `/api/position/:id`
