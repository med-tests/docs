[[Med-tests]]

##### Формат контролируемой ошибки
`status 200`

``` json
{
	"error": true, // обязательное
	"message": "<Описание ошибки, которое будет выведено в тостер>", // обязательное
	"error_code": 1,
}
```

error_code 1 - ошибка авторизации (устарел токен, пользователь не найден, нет токена)

##### Запросы
- [[Логин]] `/api/login`
- [[Регистрация]]  `/api/register`
- [[Получить список всех показателей]]  `api/get-tests`
- [[Добавить показатель]] `api/add-test`
- [[Редактировать показатель]]  `/api/edit-test/:id`
- [[Удалить показатель]] `/api/delete-test/:id`
- [[Изменить порядковый номер показателя в панели управления и в графиках анализов]] `/api/position/:id`
