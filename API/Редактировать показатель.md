`PATCH` `/api/edit-test/:id`
**id** - id показателя, целое положительное число

**Описание:** Изменяет показатель и его результаты. Требует аутентификацию.
**Тело запроса (application/json):**
``` json
{
	"id": 16, // обязательное
    "title":"Гемоглобин, Hb 123", // обязательное
    "normalFrom":1,
    "normalTo":12,
    "results": [ // может быть пустым
      { 
          "id":35,
          "date":"2025-11-14",
          "value":14
      }
    ]
}
```

| Поле           | Тип                   | Описание                                                                                                               | Обязательное | Возможные значения                                       |
| -------------- | --------------------- | ---------------------------------------------------------------------------------------------------------------------- | ------------ | -------------------------------------------------------- |
| **id**         | число                 | уникальный идентификатор показателя                                                                                    | да           | Целое положительное число                                |
| **title**      | строка                | пользовательское название показателя. Не больше 45 символов                                                            | нет          | "Гемоглобин, Hb"                                         |
| **isHidden**   | число                 | скрыт или отображен показателя в списке графиков                                                                       | нет          | 0 - показать<br>1 - скрыть                               |
| **normalFrom** | число / пустая строка | пользовательская нижняя граница нормы                                                                                  | нет          | Положительное целое или дробное число или пустая сторока |
| **normalTo**   | число / пустая строка | пользовательская верхняя граница нормы                                                                                 | нет          | Положительное целое или дробное число или пустая сторока |
| **showFrom**   | строка                | дата для фильтрации результатов в графике                                                                              | нет          | строка в формате 'YYYY-MM-DD' или пустая строка          |
| **showTo**     | строка                | дата для фильтрации результатов в графике                                                                              | нет          | строка в формате 'YYYY-MM-DD' или пустая строка          |
| **results**    | массива               | Массив с результатами анализов. Если есть изменения в результатах анализа - добавление или редактирование существующих | нет          | Массив с результатами анализов или пустой                |

| Поле       | Тип    | Описание                                    | Обязательное                                                 | Возможные значения                              |
| ---------- | ------ | ------------------------------------------- | ------------------------------------------------------------ | ----------------------------------------------- |
| **id**     | число  | уникальный идентификатор результата анализа | да - для редактирования или удаления<br>нет - для добавления | Целое положительное число                       |
| **date**   | строка | пользовательская дата сдачи анализа         | да                                                           | строка в формате  'YYYY-MM-DD'                  |
| **value**  | число  | Пользовательский результат анализа          | да                                                           | Целые и дробные числа положительные числа или 0 |
| **status** | число  | удален или нет результат анализа            | нет                                                          | 0 - удален<br>1 - активен                       |

**Успешный ответ (200):**
``` json
{
	"id": 9,
    "title": "Гемоглобин, Hb 123",
    "normalFrom": 1,
    "normalTo": 12,
    "isHidden": 0,
    "showFrom": "2025-11-14",
    "showTo": "2025-11-14",
    "position": 2,
    "results": [
        {
            "id": 8,
            "date": "2025-11-14",
            "value": 14
        }
    ]
}
```

| Поле           | Тип                   | Описание                                           | Обязательное? | Дефолтное значение                                                        | Возможные значения                                       |

| **normalTo**   | число / пустая строка | пользовательская верхняя граница нормы             | да            | ''                                                                        | Положительное целое или дробное число или пустая сторока |
| **position**   | число                 | порядковый номер для отрисовки в панели управления | да            | На 1 больше, чем количество не-удаленных показателей данного пользователя | Целое положительное число                                |

| **results**    | массив                | Результаты анализов                                | да            | пустой массив                                                             | Массив с результатами анализов или пустой                |
