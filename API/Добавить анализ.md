`POST` `api/add-test`
**Описание:**  Создает новый анализ и возвращает его. Требует аутентификацию.


**Тело запроса (application/json):**
``` json
  {
    "title":"Гемоглобин, Hb",
    "position": 10,
    "normalFrom":111,
    "normalTo":143,
    "results": [
      {"date":"2025-11-13","value":143}
    ]
  }
```

| Поле           | Тип                   | Описание                                         | Обязательное | Дефолтное значение                                                     | Возможные значения                                       |
| -------------- | --------------------- | ------------------------------------------------ | ------------ | ---------------------------------------------------------------------- | -------------------------------------------------------- |
| **title**      | строка                | пользовательское название анализа                | да           | -                                                                      | "Гемоглобин, Hb"                                         |
| **position**   | число                 | порядковый номер для отрисовки в списке анализов | да           | На 1 больше, чем количество не-удаленных анализов данного пользователя | Целое положительное число                                |
| **normalFrom** | число / пустая строка | пользовательская нижняя граница нормы            | да           | ''                                                                     | Положительное целое или дробное число или пустая сторока |
| **normalTo**   | число / пустая строка | пользовательская верхняя граница нормы           | да           | ''                                                                     | Положительное целое или дробное число или пустая сторока |
| **results**    | массива               | Массив с результатами анализа                    | да           | пустой массив                                                          | Массив с результатами анализа или пустой                 |

| Поле      | Тип    | Описание                                    | Обязательное | Дефолтное значение  | Возможные значения                              |
| --------- | ------ | ------------------------------------------- | ------------ | ------------------- | ----------------------------------------------- |
| **date**  | строка | пользовательская дата сдачи анализа         | да           | -                   | строка в формате  'YYYY-MM-DD'                  |
| **value** | число  | Пользовательский результат анализа          | да           | -                   | Целые и дробные числа положительные числа или 0 |



**Успешный ответ (200):**
``` json
{
  "id":16, // обязательное
  "title":"Гемоглобин, Hb", // обязательное
  "position": 12, // обязательное
  "normalFrom":111,
  "normalTo":143,
  "isHidden":0, // при создании всегда 0
  "showFrom":"2025-11-15",
  "showTo":"2025-11-23",
  "results": [ // может быть пустым
    {"id":35,"date":"2025-11-13","value":143}
  ]
}
```

| Поле           | Тип                   | Описание                                         | Обязательное? | Дефолтное значение                                                     | Возможные значения                                       |
| -------------- | --------------------- | ------------------------------------------------ | ------------- | ---------------------------------------------------------------------- | -------------------------------------------------------- |
| **id**         | число                 | уникальный идентификатор анализа                 | да            | Автоинкремент от БД                                                    | Целое положительное число                                |
| **title**      | строка                | пользовательское название анализа                | да            | -                                                                      | "Гемоглобин, Hb"                                         |
| **normalFrom** | число / пустая строка | пользовательская нижняя граница нормы            | нет           | ''                                                                     | Положительное целое или дробное число или пустая сторока |
| **normalTo**   | число / пустая строка | пользовательская верхняя граница нормы           | нет           | ''                                                                     | Положительное целое или дробное число или пустая сторока |
| **position**   | число                 | порядковый номер для отрисовки в списке анализов | да            | На 1 больше, чем количество не-удаленных анализов данного пользователя | Целое положительное число                                |
| **isHidden**   | число                 | скрыт или нет анализ из графиков                 | да            | 0                                                                      | 0 - показан<br>1 - скрыт                                 |
| **showFrom**   | строка                | дата для фильтрации результатов в графике        | нет           | дата самого раннего результата анализа                                 | строка в формате 'YYYY-MM-DD' или пустая строка          |
| **showTo**     | строка                | дата для фильтрации результатов в графике        | нет           | дата самого последнего результата анализа                              | строка в формате 'YYYY-MM-DD' или пустая строка          |
| **results**    | массив                | Результаты анализа                               | да            | пустой массив                                                          | Массив с результатами анализа или пустой                 |

| Поле      | Тип    | Описание                                    | Обязательное | Дефолтное значение  | Возможные значения                              |
| --------- | ------ | ------------------------------------------- | ------------ | ------------------- | ----------------------------------------------- |
| **id**    | число  | уникальный идентификатор результата анализа | да           | Автоинкремент от БД | Целое положительное число                       |
| **date**  | строка | пользовательская дата сдачи анализа         | да           | -                   | строка в формате  'YYYY-MM-DD'                  |
| **value** | число  | Пользовательский результат анализа          | да           | -                   | Целые и дробные числа положительные числа или 0 |
