`POST` `/api/login`

**Описание:** обменивает логин и пароль на jwt-token

**Тело запроса**
``` json
{
	"login": "login", // обязательный, уникальный
	"password": "Password1!" // обязательный
}
```


**Успешный ответ (200):**
``` json
{
	"token": "<token>"
}	
```

**Ошибки**
- Если нет логина или пароля => 'Недостаточно данных'
- Если не найден юзер по логину => 'Неверный логин или пароль'
- Если пароль не совпал => 'Неверный логин или пароль'
