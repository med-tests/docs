`PATCH` `/api/position/:id`
**id** - id анализа, целое положительное число

**Описание:** обновляет `position` показателя и всех затронутых перемещением показателей. Возвращает id показателя.

**Тело запроса**
``` json
{
	"newPosition": 10, // обязательное
	"oldPosition": 12, // обязательное
}
```

**Успешный ответ (200):**
``` json
{
	"id": 9
}
```
