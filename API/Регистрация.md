`POST` `/api/register`

**Описание:**  регистрирует нового пользователя

**Тело запроса**
``` json
{
	"login": "login", // обязательный, уникальный
	"password": "Password1!" // обязательный
}
```



**Успешный ответ (200):**
``` json
{
	"token": "<token>"
}	
```

**Ошибки**
- Если нет логина или пароля => 'Недостаточно данных'
- Если логин уже используется => 'Логин занят'
