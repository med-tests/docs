[[Med-tests]]
##### Показатель (test)
- **id**\* - *INTEGER* - id анализа. Устанавливается базой.
- **title**\* - *TEXT* - пользовательское название показателя
- **userId**\* - *INTEGER* - id пользователя, которому принадлежит показателя. Не отправляется на фронт, используется для поиска результатов показателя.
- **normalFrom** - *INTEGER* - пользовательское значение нижней границы нормы для показателя
- **normalTo** - *INTEGER* - пользовательское значение верхней границы нормы для показателя
- **isHidden** - *INTEGER* - определяет, отрисован график или нет. 0 - показать, 1 - скрыть.
- **showFrom** - *TEXT* - дата в формате `YYYY-MM-DD`, если есть хоть один результат или пустая строка. Определяет *с* какой даты включительно выводятся результаты показателя на графике. По умолчанию - дата самого раннего результата. Может переопределяться пользователем. Если у показателя нет результатов, будет пустой строкой.
- **showTo** - *TEXT* - дата в формате `YYYY-MM-DD`, если есть хоть один результат или пустая строка. Определяет *до* какой даты включительно выводятся результаты на графике. По умолчанию - дата самого последнего результата. Может переопределяться пользователем. Если у показателя нет результатов, будет пустой строкой.
- **results**  - *ARRAY* - массив результатов.
- **status** - *INTEGER* - Указывает, удален ли показатель. 0 - удален, 1 - активен. Не отправляется на фронт, но по нему фильтруются анализы, отправляемые на фронт.
- **position**  - *INTEGER* - порядковый номер показателя при отображении. Чем больше, тем выше.

##### Результат (анализа по показателю) (result)
- **id**\* - *INTEGER* - id результата. Устанавливается базой.
-  **userId**\* - *INTEGER* - id пользователя, которому принадлежит результат. Не отправляется на фронт, используется для поиска.
- **date**\* - *TEXT* - дата в формате `YYYY-MM-DD`. Пользовательская дата сдачи анализа.
- **value**\* - *INTEGER* - Пользовательский результат анализа. Может быть 0.
- **status** - *INTEGER* - Указывает, удален ли результат. 0 - удален, 1 - активен. Не отправляется на фронт, но по нему фильтруются результаты, отправляемые на фронт.
- **test_id**\* - *INTEGER* - id показателя, которому принадлежит результат. Не отправляется на фронт, используется для поиска.

##### Юзер (user)
- **id**\*  - *INTEGER* - id пользователя
- **login\*** - *VARCHAR(255) UNIQUE* - уникальное имя пользователя
- **password\*** - *VARCHAR(255)* - пароль
- **status** - *INTEGER* - Указывает, удален ли пользователь. 0 - удален, 1 - активен. 
- **date_create** - *DATETIME* - Таймштамп даты создания пользователя
