#### Получить список всех анализов
`GET` `/tests`
**Описание:** возвращает массив всех не-удаленных анализов
**Успешный ответ (200):**
``` json
[
  {
    "id":16, // обязательное
    "title":"Гемоглобин, Hb", // обязательное
    "normalFrom":111,
    "normalTo":143,
    "isHidden":0,
    "showFrom":"2025-11-15",
    "showTo":"2025-11-23",
    "results": [ // может быть пустым
      {"id":35,"date":"2025-11-13","value":143}
    ]
  }
]
```

#### Добавить анализ
`POST` `/add-test`
**Описание:**  Создает новый анализ и возвращает его
**Тело запроса (application/json):**
``` json
  {
    "title":"Гемоглобин, Hb", // обязательное
    "normalFrom":111,
    "normalTo":143,
    "results": [ // может быть пустым
      {"date":"2025-11-13","value":143}
    ]
  }
```


**Успешный ответ (200):**
``` json
{
  "id":16, // обязательное
  "title":"Гемоглобин, Hb", // обязательное
  "normalFrom":111,
  "normalTo":143,
  "isHidden":0, // при создании всегда 0
  "showFrom":"2025-11-15",
  "showTo":"2025-11-23",
  "results": [ // может быть пустым
    {"id":35,"date":"2025-11-13","value":143}
  ]
}
```
